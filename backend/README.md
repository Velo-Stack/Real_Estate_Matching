# Real Estate Matching Backend

Node.js + Express backend for managing real estate offers, requests, matching, teams, conversations, notifications, and reports.

## Tech Stack
- Node.js / Express
- PostgreSQL
- Prisma ORM
- JWT authentication
- Socket.IO

## Main Scripts
- `npm run dev`: run in development mode
- `npm start`: run in production mode

## Environment
Create `backend/.env` with at least:
- `DATABASE_URL`
- `JWT_SECRET`
- `PORT` (optional)
- `PUBLIC_BASE_URL` (optional, used for submission links)
